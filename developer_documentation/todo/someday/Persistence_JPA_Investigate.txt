EclipseLink Setup:

http://wiki.eclipse.org/EclipseLink/Installing_and_Configuring_EclipseLink

Download the EclipseLink jars from http://www.eclipse.org/eclipselink/downloads/
Unpack and move eclipselink folder to somewhere eg. under Eclipse application.
mv -i ~/Downloads/eclipselink  /Applications/eclipse/

Set the env path in .bashrc (or whatever shell you use)

export ECLIPSELINK_HOME=/Applications/eclipse/eclipselink
